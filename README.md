# Fire Force Anime Website

A simple static website inspired by **Fire Force (Enen no Shouboutai)**, built using **HTML, Bootstrap 5, and JavaScript**.  
This project focuses on clean layout, responsive design.

---

## Features

- Responsive layout for mobile and desktop devices
- Fixed navigation bar using Bootstrap
- Section-based navigation (Home, About, Episode, Anime)
- Using API : https://api.jikan.moe/v4/anime/38671/episodes
- Dynamic episode list loaded with JavaScript
- Google Fonts integration (Poppins)
